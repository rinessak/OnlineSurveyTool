# OnlineSurveyTool
Backend application that allows users to create surveys, submit responses, and store results using RESTful APIs. Built with Node.js and MongoDB to handle dynamic survey structures and response data.

## Tech Stack

- Node.js

- Express.js

- MongoDB

- Mongoose

- REST API

## Features

- Create, update, and delete surveys

- Define surveys with dynamic questions and answer options

- Submit and store user responses

- Server-side validation for incoming requests

- Organized backend structure using routes, controllers, and models

  ## Project Structure
  
/routes      → API route definitions  
/controllers → Request handling logic  
/models      → Mongoose schemas  

## Setup Instructions

1. Clone the repository

2. Run npm install

3. Create a .env file and add your MongoDB connection string

4. Run nodemon index.js
